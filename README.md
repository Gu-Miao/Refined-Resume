# 项目名称：
```
Refined Resume
```
# 成员：
```
谷淼 刘京 张高翔
```
# 简介：
```
在线制作电子版个人简历平台网站
```
# 功能：
```
游客登录
登录注册(表单验证)
个人信息修改（表单验证，如姓名过长）
根据模板在线制作个人简历功能（提供模板，颜色，风格，样式）
自制个人简历功能（提供元素，拖拽）
简历管理功能（增，删，改）
简历下载功能（格式转换）
简历模板收藏功能
自制模板分享功能
```
# 用户面向
```
对个人简历有需求的求职者或者个人
```
